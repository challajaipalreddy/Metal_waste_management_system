# User Data Directory

This directory contains user registration and profile information for the Metal Waste Exchange platform. The directory is organized into three main categories:

1. `regular_users/` - Stores information for standard users who want to sell or donate metal waste
2. `scrap_processors/` - Contains data for registered scrap processing companies
3. `buyers/` - Maintains information for verified metal waste buyers

Each subdirectory contains user-specific data including:
- Registration details
- Profile information
- Verification status
- Activity history

## Security Notice
- All user data is encrypted at rest
- Access to this directory is restricted to authorized personnel only
- Regular backups are maintained
- Personal information is handled in compliance with data protection regulations
