# Buyers Directory

This directory stores information for verified metal waste buyers on the Metal Waste Exchange platform.

## Data Structure
- Buyer profiles
- Business information
- Purchase history
- Payment records
- Verification documents
- Business licenses

## File Naming Convention
Files are named using the format: `buyer_[company_id].json`

## Data Fields
- Buyer ID
- Company Name
- Business Registration Number
- Business Address
- Contact Person
- Email
- Phone Number
- Business Type
- Verification Status
- Payment Methods
- Purchase History
- Credit Rating
- Preferred Metal Types
- Business License Number
- Tax Information
