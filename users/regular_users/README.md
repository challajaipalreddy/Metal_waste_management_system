# Regular Users Directory

This directory stores information for standard users of the Metal Waste Exchange platform who want to sell or donate metal waste.

## Data Structure
- User profiles
- Registration information
- Transaction history
- Verification documents
- Contact details

## File Naming Convention
Files are named using the format: `user_[unique_id].json`

## Data Fields
- User ID
- Full Name
- Email
- Phone Number
- Address
- Registration Date
- Verification Status
- Transaction History
- Preferred Payment Method
- Account Status
