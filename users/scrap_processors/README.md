# Scrap Processors Directory

This directory contains information for registered scrap processing companies on the Metal Waste Exchange platform.

## Data Structure
- Company profiles
- Business registration details
- Processing capabilities
- Certification documents
- Operating licenses
- Facility information

## File Naming Convention
Files are named using the format: `processor_[company_id].json`

## Data Fields
- Company ID
- Business Name
- Registration Number
- Business Address
- Contact Person
- Email
- Phone Number
- Processing Capabilities
- Certification Status
- Operating Hours
- Facility Location
- Equipment List
- Environmental Compliance Status
- Insurance Information
